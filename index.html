                <h2 class="text-3xl font-bold mb-4">Core Concepts</h2>

                <p class="mb-8 text-gray-600">Before diving into atomic models, it's crucial to understand the terminology used to classify different atoms and ions based on their subatomic particles. This section defines these fundamental classifications.</p>

                <div class="grid grid-cols-1 md:grid-cols-2 gap-6">

                    <div class="info-card p-6 rounded-lg">

                        <h3 class="font-bold text-xl mb-2 accent-text">Isotopes</h3>

                        <p>Atoms with the same number of protons (same Z) but a different number of neutrons (different A).</p>

                        <p class="mt-2 text-sm text-gray-500">Example: Protium (¹H), Deuterium (²H), Tritium (³H)</p>

                    </div>

                    <div class="info-card p-6 rounded-lg">

                        <h3 class="font-bold text-xl mb-2 accent-text">Isobars</h3>

                        <p>Atoms of different elements with the same mass number (A) but different atomic numbers (Z).</p>

                        <p class="mt-2 text-sm text-gray-500">Example: ¹⁴C and ¹⁴N</p>

                    </div>

                    <div class="info-card p-6 rounded-lg">

                        <h3 class="font-bold text-xl mb-2 accent-text">Isotones</h3>

                        <p>Atoms of different elements with the same number of neutrons (A-Z).</p>

                        <p class="mt-2 text-sm text-gray-500">Example: ¹³C (7n) and ¹⁴N (7n)</p>

                    </div>

                    <div class="info-card p-6 rounded-lg">

                        <h3 class="font-bold text-xl mb-2 accent-text">Isoelectronic Species</h3>

                        <p>Atoms or ions that have the same number of electrons.</p>

                        <p class="mt-2 text-sm text-gray-500">Example: N³⁻, O²⁻, Ne, Na⁺ (all have 10 electrons)</p>

                    </div>

                    <div id="thomson-model" class="model-content">

                        <h3 class="text-2xl font-semibold mb-2">Thomson's "Plum Pudding" Model (1897)</h3>

                        <p class="mb-4">Proposed the atom was a uniform sphere of positive charge with negatively charged electrons embedded within it, much like plums in a pudding.</p>

                        <h4 class="font-bold">Key Limitations:</h4>

                        <ul class="list-disc list-inside text-gray-600">

                            <li>Could not explain the results of Rutherford's gold foil experiment.</li>

                            <li>Did not account for the neutrality of the atom in a stable way.</li>

                        </ul>

                    </div>

                    <div id="rutherford-model" class="model-content hidden">

                        <h3 class="text-2xl font-semibold mb-2">Rutherford's Nuclear Model (1911)</h3>

                        <p class="mb-4">Based on his α-particle scattering experiment, Rutherford proposed a model where the atom has a tiny, dense, positively charged nucleus at the center, with electrons orbiting it.</p>

                        <h4 class="font-bold">Key Limitations:</h4>

                        <ul class="list-disc list-inside text-gray-600">

                            <li>Violated classical physics: An orbiting electron should continuously lose energy and spiral into the nucleus.</li>

                            <li>Could not explain the discrete line spectra of atoms.</li>

                        </ul>

                    </div>

                    <div id="bohr-model-desc" class="model-content hidden">

                        <h3 class="text-2xl font-semibold mb-2">Bohr's Planetary Model (1913)</h3>

                        <p class="mb-4">Bohr modified Rutherford's model by proposing that electrons exist in specific, quantized orbits with fixed energy levels. Electrons only absorb or emit energy when they jump between these orbits.</p>

                        <h4 class="font-bold">Key Limitations:</h4>

                        <ul class="list-disc list-inside text-gray-600">

                            <li>Only worked perfectly for single-electron species (like H, He⁺).</li>

                            <li>Could not explain the Zeeman effect or Stark effect (splitting of spectral lines).</li>

                            <li>Violated the Heisenberg Uncertainty Principle.</li>

                        </ul>

                    </div>

                    <div id="quantum-model-desc" class="model-content hidden">

                        <h3 class="text-2xl font-semibold mb-2">Quantum Mechanical Model (1926)</h3>

                        <p class="mb-4">The modern model treats electrons as waves, based on work by de Broglie and Schrödinger. It does not define an exact path for an electron but describes a region of probability called an orbital where the electron is likely to be found.</p>

                        <h4 class="font-bold">Key Features:</h4>

                        <ul class="list-disc list-inside text-gray-600">

                            <li>Based on probability, not certainty.</li>

                            <li>Introduced the concept of quantum numbers to describe an electron's state.</li>

                            <li>Successfully explains the behavior of multi-electron atoms.</li>

                        </ul>

                    </div>

                </div>

                <h2 class="text-3xl font-bold mb-4">Bohr's Model & Hydrogen Spectrum</h2>

                <p class="mb-8 text-gray-600">Bohr's model was a critical step towards quantum mechanics, successfully explaining the observed line spectrum of hydrogen. It proposed that when an electron jumps from a higher energy orbit ($n_2$) to a lower one ($n_1$), it emits a photon with a specific wavelength, creating a spectral line.</p>

                        <p class="mb-4">Select the initial and final energy levels for an electron transition in a Hydrogen atom (Z=1) to see the resulting spectral line and its properties.</p>

            document.getElementById('element-config-btn').addEventListener('click', function() {

                const Z = parseInt(elementSelect.value);

                let electrons = Z;

                let config = {};

                if (Z === 24) { // Cr exception

                    config = {'1s':2, '2s':2, '2p':6, '3s':2, '3p':6, '4s':1, '3d':5};

                } else if (Z === 29) { // Cu exception

                    config = {'1s':2, '2s':2, '2p':6, '3s':2, '3p':6, '4s':1, '3d':10};

                } else {

                    for(const orb of orbitalOrder) {

                        if (electrons <= 0) break;

                        const capacity = orbitalCapacity[orb.slice(-1)];

                        const fill = Math.min(electrons, capacity);

                        config[orb] = fill;

                        electrons -= fill;

                    }

                }

               

                let configString = Object.entries(config).map(([orb, el]) => `${orb}<sup>${el}</sup>`).join(' ');

                let unpaired = 0;

                if (Z === 24) unpaired = 6;

                else if (Z === 29) unpaired = 1;

                            unpaired = lastOrbElectrons;

                        } else {

                            unpaired = orbCapacity - lastOrbElectrons;

                        }

                    }

                }
